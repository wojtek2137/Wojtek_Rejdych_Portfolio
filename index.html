About Me

Wojtek Rejdych -

Front End Web Developer & Saxophonist

I'm a graduate of the Frederic Chopin Music School in Krakow. Participant of many jazz courses and workshops, incl. International Cho-Jazz Workshop in Chodzież. I had an opportunity to learn from such saxophonists as Dayna Stephens, Roxy Coss, Rosalio Giuliani, Jerzy Główczewski and Maciek Sikała. Currently I'm a student at the Krzysztof Penderecki Academy of Music in Krakow in the jazz saxophone class under the supervision of prof. dr hab. Ryszard Krawczuk and dr Marcin Slusarczyk


My lifelong passion is playing the saxophone, but also programming. I see many analogies in these two (seemingly different) areas. The programmer is also an artist. Only I know how much time and energy I've devoted to be accepted for the very difficult jazz studies. I have achieved this goal, but I still want to develop.


Currently, my next dream is to develop skills and work in the IT industry. With great persistence, but also with the pleasure of learning "new things", I devote a few hours a day to learning programming. I think I am ready to take the next step - to start an internship because I know that the best teacher is practice.

My Main Projects

Jazz Scale

Generator

VexFlow JS

VexFlow is an open-source online music notation rendering API. It is written completely in JavaScript, and runs right in the browser.

HTML-5 / CSS3 / JavaScript ES6

I used these technologies to improve bootstrap classes for my project, but also my entire algorithm is written in Vanilla JS ES6

Bootstrap 4

I used bootstrap because the most important goal was to "do magic behind the scenes" in JavaScript

Description

This is the app I am most proud of, but it has also taken me the most time. I decided to do something that will result from the merger of the two worlds that I deal with.
So I used my academic music knowledge and programming skills so I was able to create an algorithm that generates a scale in each musical key, only with a numerical recipe for a given scale stored in an one and only one array of numbers.
This algorithm is so universal that it allows me to easily add new scales thanks to one short array.
To write what my algorithm generates, I used Vexflow JS , which was created for MANUAL music notation. Phew, it took me a long time to make working colaboration of my algorithm and VexFlow JS... but in the end THAT'S WORKING!
I am going to add many more scales in the future and I think it could be a great program to learn jazz harmony for students of this very difficult music.
